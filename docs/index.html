About BLT-Lettuce

BLT-Lettuce is a Slack bot developed by the OWASP BLT team for the OWASP Slack workspace. It welcomes new members, provides helpful information, and assists the community with various tasks.

The bot is powered by a Cloudflare Python Worker that handles webhooks and tracks statistics displayed on this dashboard.

Learn more about the project on our GitHub repository.
